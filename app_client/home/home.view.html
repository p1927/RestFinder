RestFinder

 Find places to work with wifi near you!

Restaurant Finder provides an easy way to find the best food offers and places to eat near you. If you allow it, the app can find the nearest restaurants just by using your location. However, if you already have a place in mind, type in the address, the name or the zip code and let it work for you!

This Web application is made over MEAN stack. MongoDB, Express, Jade and NodeJS. Angular components are to be added soon.The website allows user to find Resturants nearby. And sorts it out as per increasing distance from user. This app uses Google Geolocation API.

Functionality to be added :

  • Comments, Replies, Search functions
  • Filtering places based on different criterias.
