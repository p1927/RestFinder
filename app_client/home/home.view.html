RestFinder

 Find places to work with wifi near you!

Restaurant Finder provides an easy way to find the best places to eat near you. If you allow it, the app can find the nearest restaurants just by using your location. However, if you already have a place in mind, type in the address or the name and let it work for you!

This Web application is made over MEAN stack.
  • MongoDB : Document Database
  • Express : Web Application Framework
  • AngularJS : Front-End Web Application Framework
  • NodeJS : Software Platform
  • A web application to find restaurants nearby the user location.
  • Uses HTML5 Geo-location features to get user latitude and longitudes.
  • Uses Google Directions API for providing the directions to the restaurant.
  • Built on MVC architecture.
  • Responsive Web Design, Single page application made using AngularJS routing.
    •
    Features :
  • Find nearby restaurants based on location and maximum distance allowed by user.
  • Filter restaurants based on facilities, name, distance, address.
  • Provides directions from user location to restaurant.
  • Users can Login and add reviews, comments and upvote.
  • User authorization using Google and Facebook.
    •
  • Implements RESTful- API supporting GET, PUT, POST, DELETE methods for restaurants, reviews, comments, users.
  • User-API for User Information and user authentication using JSON Web Token and OAuth.
